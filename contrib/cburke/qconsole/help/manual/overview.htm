Overview

Qconsole runs as a separate task from q and communicates via the usual q ipc. The q process can be anywhere on the network.

Qconsole has two window types:

  • Term - replaces the usual q console

  • Edit - a tabbed script editor and project manager. Each edit window can have any number of opened scripts. At most two edit windows may be active.
