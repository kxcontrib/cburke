Price

There are three column formats, where date is any date/time type, and size is optional:

1. date,price[,size]
   - plots price for a single stock

For example: 

date       price size
---------------------
2011.03.07 20.5  200
2011.03.09 20.75 230
2011.03.10 20.99 260
...
2. date,id1,id2,id3...
   - plots prices for stocks in columns

For example: 

date       AAPL  AIG   IBM
----------------------------
2010.10.04 26.94 29.8  36.36
2010.10.05 27.82 30.83 36.52
2010.10.06 28.02 31.48 36.91
...
3. date,sym,price
   - plots prices for stocks in sym

For example: 

date       sym  price
---------------------
2010.10.04 AMD  99.26
2010.10.04 DELL 9.93
2010.10.04 MSFT 78.24
...
