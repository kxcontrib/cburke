Open/High/Low/Close

Columns are: date,open,high,low,close[,size]

where date is any date/time type, and size is optional.

For example: 

date       open  high  low   close size
---------------------------------------
2011.03.07 25.68 26.33 24.83 24.83 200
2011.03.09 24.83 24.83 24.21 24.21 230
2011.03.10 24.21 26.07 23.28 26.07 260
...
